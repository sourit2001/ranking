# 动态排名视频生成器

## 项目简介
这是一个用Python开发的图形界面应用程序，可以将Excel数据转换成动态排名视频。主要用于展示车型销量数据的变化趋势，通过生动的动画效果直观地展示数据变化。

## 功能特点
- 支持Excel(.xlsx, .xls)和CSV文件格式
- 拖拽上传文件功能
- 自定义视频标题
- 显示前20名排名
- 实时显示生成进度
- 大字体清晰显示时间点
- 显示总销量统计
- 美观的图形界面
